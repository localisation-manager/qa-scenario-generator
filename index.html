Set up a Slack Workflow to automatically ping your QA team every time a feature announcement is posted — with a direct link to this tool.

Slack Workflow Builder — Step by Step
1

Open Workflow Builder

In Slack, click your workspace name → Tools & settings → Workflow Builder. Click New Workflow.

2

Set the Trigger

Choose "When a message matches a search" as the trigger type. Set the channel to #fresha_team. Add keyword: FEATURE ANNOUNCEMENT.

3

Add a "Send a message" step

Choose your QA channel (e.g. #qa-scenarios) as the destination. Use this message template:

🧪 New feature announcement detected in #fresha_team! *Feature:* {{message.text | truncate: 80}} *Posted by:* {{message.sender}} → Open QA Generator: https://claude.ai/... React with ✅ once scenarios are saved to Notion.
4

Replace the link

Swap the placeholder URL with the actual link to this tool. You can bookmark this Claude conversation or save the downloaded HTML file to a shared location.

5

Publish & test

Click Publish. Post a test message containing "FEATURE ANNOUNCEMENT" in the channel to verify the notification fires.
